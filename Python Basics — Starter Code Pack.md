# Python Basics — Starter Code Pack (ForgeLabs)

Welcome to the official starter pack for the *Python Basics* tutorial.

This folder contains:
- Basic Python scripts covered in the video
- A mini project (Age Calculator)
- Extra learning resources

## Files
### 01_print_and_comments.py
Learn how to use print statements and comments.

### 02_variables.py
Introduction to variables and assignments.

### 03_data_types.py
Breakdown of Python’s core data types: int, float, str, bool.

### 04_age_calculator_project.py
A beginner-friendly mini project you can modify.

## How to Use
1. Install Python from https://python.org  
2. Open any file in VS Code or IDLE  
3. Press **Run**  
4. Modify the code and experiment!

## Bonus Resources
Check the `bonus_resources` folder for:
- Python cheat sheet (quick revision)
- Variable naming rules
